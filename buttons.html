Buttons

Buttons are interactive elements which can be used to trigger events such as submitting a form or redirecting the user to another page.

Overview

Within the simplePattern library, there are many variations of buttons available, including filled, outlined, text, grouped, dropdown and disabled.

The .btn class can be used with several types of HTML elements.

Anchor 
<a class="btn">Anchor</a>
<button class="btn">Button</Button>
<input type="submit" class="btn"/>
<input type="reset" class="btn"/>

Generic buttons

Generic buttons are those that are used to submit forms, cancel actions, redirect the user to another page or to trigger custom event.

The generic buttons are available in three different styles: filled, outlined and basic text.

Each style has 4 colours, including grey for the default style, blue for the important CTA buttons, yellow to represent a button which would cause notable changes to a system; and red reprets a button that when pressed, will result in irreversible changes.

Dropdown button

simplePatterns dropdown button uses JavaScript to display a dropdown list when the button clicked. This can be used to display a list of form options, links or actions.

The dropdown button uses the same .btn classes and therefore can be outlined, filled or text-based, as well as be different colours.

Option 1 Option 2 Option 3
Option 1 Option 2 Option 3

  <a class="btn filled">Dropdown<i class="fas fa-angle-down"<i></a>

      <a class="drop_down_option">Option 1</a>
      <a class="drop_down_option">Option 2</a>

</div>

Disabled Button

The disabled button is used to represent where a button usually is, but due to a particular reason, perhaps the user not entering the correct information, is therefore disabled.

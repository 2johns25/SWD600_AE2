Jumbotron

Jumbotrons are used to present the user with information to get them to interact with a call to action.

Overview

simplePattern's takes up the full width of it's container and contains a CTA button. The Jumbotron can also contain other elements such as headers, paragraphs, line breaks and so on.

Example

Jumbotron header

This is a jumbotron component

<div class="jumbotron">
    <div class="header_text_container">
    <h1>Jumbotron header</h1>
    <p>This is a jumbotron component</p>
    <a class="btn filled">Default<a>
</div>

Grid

Grids are used for maintaining consistency within a website and ensure items are aligned at different sized viewports.

Overview

simplePattern's grid is implemented using basic floats. The grid content must be contained within a grid container, which contains rows. Individual columns must be the direct children of rows.

Classes

There are three groups of classes that correspond with three different viewport widths, which are distinguishable with the -sm- -md- and -wd- classes.

-sm- classes are used for screen widths between 320px and 600px.

-md- classes are used for screen widths between 600px and 768px.

-wd- classes are used for screen widths higher than 768px.

Looking at the example code below, the div will take up 12 columns between the screen sizes of 320px and 600px, 6 columns between 600px and 768px and 12 columns at any screen width larger than 768px. 

<div class="grid">
    <div class="row">
        <div class="col-sm-12 col-md-6 col-wd-12">
            <!-- Custom HTML goes here -->
        </div>  
    </div>  
</div>

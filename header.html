Header

Headers are located at the top the page, underneath the navigation header and are used to provide important information to the user.

Overview

simplePattern's responsive header comes in three versions: default, light and dark, each of which can contain text which is centered horizontally and vertically.

Examples

Default

The default header styles includes a single image, with optional text but no overlay. This is great for when you have naturally contrasting images and text.

Secondary header

Dark overlay

The header with the dark overlay, will enable you to use any picture with light text. Simply add the .dark class and the .light_text class as seen below.

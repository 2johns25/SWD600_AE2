Navigation

The navigation sits at the top of the website and is home to the sites logo and links to important sections of the site.

Overview

simplePattern's navigation comes with a full width, horizontal, responsive navigation bar. Each link comes with a :hover and :active effects, as well as a .active_nav class to style the nav link for the current page that is being displayed.
